# University
This Java programm is about the class inharitance. It's an University program which has Students who have many info about Courses. There are two types of Students.
The simple Student who has a name, an ID and attends some Courses and the Graduate Student who has also name,ID and Courses but they alse have a Supervisor. 
Both of them can print their Info and the users can opt the way that they enter the data, randomly or manually.
